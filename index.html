o que é display flex ?

O display flex é uma propriedade do CSS que cria layouts flexíveis e responsivos, organizando elementos em linha ou coluna e facilitando o alinhamento e a distribuição do espaço.

o que é justifycontent ?

justify-content é uma propriedade do CSS usada com flexbox para alinhar e distribuir os itens ao longo do eixo principal, podendo posicioná-los no início, fim, centro ou com espaços entre eles.

o que é align-items ?

align-items é uma propriedade do CSS usada no flexbox para alinhar os itens no eixo transversal, podendo posicioná-los no início, fim, centro, esticados ou pela linha de base.
